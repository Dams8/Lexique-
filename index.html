Lexique de développeur web

A

AJAX
Ce terme désigne une technologie utilisée dans le domaine de la création de sites web. ..
API
Interface de programmation d'application, permettant à différentes applications de communiquer entre elles.
Angular
Framework JavaScript open-source pour la création d'applications web à page unique (SPA).
Animate.CSS
Animate.CCS esr un framework qui permet de trouver des animations

B

Bigdata
Le Big Data, c’est l’ensemble massif de données disponible grâce à toutes les applications numériques. Le Big Data permet, aujourd’hui, l’analyse en tant réel et la prise de décision à travers des statistiques. Tout ceci est possible grâce à la numérisation de tout ou partie des processus analogiques.
Bootstrap
Framework CSS open-source pour la création de sites web responsives et mobiles-first.

C

CSS
Cascading Style Sheets, langage de feuilles de style utilisé pour styliser les pages web.

J

JavaScript
Langage de programmation de scripts utilisé pour rendre les pages web interactives.
jQuery
Bibliothèque JavaScript open-source facilitant la manipulation du DOM, les animations, les requêtes AJAX, etc.

R

React
Librairie JavaScript open-source pour la création d'interfaces utilisateur.
